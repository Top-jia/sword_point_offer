	@1.二维数组
	在一个二维数组中, 每一行都按照从左至右递增的顺序排列, 每一列都按照从上到下递增的顺序排列, 请完成一个函数, 输入一个二维数组和一个证整数, 判断数组中是否包含该整数.
	code1.cpp

	例如:
		输入数组:
		1, 5, 7, 9,
		2, 6, 8, 10,
		7, 9, 10, 12
		输入整数值: 6
	解决思路:
		先确定最大行或者最大列来固定一方, 用一个变量来控制另一方, 从而确定这个值的存在是否.
	总结:
		主要难点在于对列值进行控制, 自己没有反应出来. 还是必须动笔一步步来画整个过程.
